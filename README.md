<h2>Installation Steps:</h2>

<p>1. Node.js is required! So install that.</p>

<p>2. install Express</p>

```
npm install express
```

<p>3. install nodemon (optional)</p>

```
npm install nodemon
```

<p>4. run the thing</p>

```
npm start
```
